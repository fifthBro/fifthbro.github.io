Overview

Preliminary integration that displays Android Auto navigation instructions from Porsche PCM5 / VW / Audi MH2P to instrument cluster displays to instrument cluster displays. Translates Android Auto DSI events into BAP (Board Access Protocol) messages for real-time turn-by-turn navigation on your cluster.

✅ Status: Beta (v01010)

Why This Project?

Native navigation monopolizes the instrument cluster, this code will change that!. This integration:

  • Intercepts Android Auto navigation events via DSI (Device System Interface)
  • Converts them to BAP protocol messages
  • Sends to cluster via CombiBAPServiceNavi
  • Overrides native navigation with 2-second heartbeat
  • Supports both Google Maps and Waze

Features

🗺️

Full Navigation Support

Turn arrows, distance, road names, roundabouts, highway exits - everything from Android Auto displayed on your cluster.

⚡

Real-Time Updates

60 FPS updates with automatic heartbeat to override native navigation. Seamless transitions and no flickering.

🌍

Intelligent Unit Detection

Automatic km/mi and LHD/RHD detection via DSI services with JSON country fallback. Supports 60+ countries.

💾

Automatic Storage Mounting

Auto-remounts SD/USB storage as writable via ASI services for external logging. No manual intervention needed.

🔧

JSON Configuration

Runtime-configurable logging, heartbeat, units, and more via embedded JSON config. No recompilation required.

📊

Comprehensive Logging

Detailed file logging with configurable size limits, external storage support, and optional message hashing for privacy.

Architecture

Component Overview

Android Auto (Phone)
        ↓
    DSI Events (Navigation, Media)
        ↓
AndroidAutoClusterIntegration
    ├─ Event Processing
    ├─ Unit Conversion (km/mi)
    ├─ BAP Message Generation
    └─ Heartbeat Management
        ↓
CombiBAPServiceNavi (OSGi)
        ↓
MOST Bus → Instrument Cluster

Key Components

AndroidAutoClusterIntegration

Core integration class that implements DSIAndroidAuto2ListenerSafe to receive Android Auto events:

  • Navigation Events: Turn info, distance, route guidance status
  • BAP Translation: Converts AA event codes to BAP mainElement/direction
  • Heartbeat: 2-second timer continuously sending BAP messages to override native nav
  • Configuration: Loads JSON config embedded in JAR with runtime overrides

AndroidAuto2Subsystem

Manages lifecycle and OSGi service discovery:

  • Creates AndroidAutoClusterIntegration with StorageMountHandler
  • Looks up CombiBAPServiceNavi, ISysServices, ICarCoreServices
  • Registers integration with AndroidAuto2ListenerDistributor
  • Initializes StorageMountHandler BEFORE logging starts

BAP Protocol Mapping

Android Auto Event → BAP Message
─────────────────────────────────
TURN_LEFT (4, side=1)    → mainElement=2 (TURN), direction=1 (LEFT)
SLIGHT_RIGHT (3, side=2) → mainElement=2 (TURN), direction=4 (SLIGHT_RIGHT)
ROUNDABOUT (11-13)       → mainElement=3 (ROUNDABOUT), direction=0 (STRAIGHT)
ON_RAMP (7)              → mainElement=4 (HIGHWAY), direction=left/right
DESTINATION (19)         → mainElement=6 (DESTINATION)

Distance updates         → updateDistanceToNextManeuver(distance, unit, bargraph)
Road names               → updateTurnToInfo(roadName, signpost)

Configuration

Configuration is embedded in the JAR at androidauto_cluster_config.json. Edit and recompile to change settings:

Config File

{
  "buildVersion": "__VERSION__",
  "config": {
    "enableFileLogging": true,
    "enableFileHashing": true,
    "enableExternalLogging": true,
    "enableRoundaboutLatching": true,
    "enableHeartbeat": true,
    "forceImperial": false,
    "forceRHD": false,
    "logFilePath": "/tmp/androidauto_cluster.log",
    "hashedLogFilePath": "/tmp/androidauto_cluster_hashed.log",
    "logFileSize": 1
  },
  "countries": [
    { "code": "US", "rhd": false, "imperial": true },
    { "code": "GB", "rhd": true, "imperial": true },
    { "code": "DE", "rhd": false, "imperial": false }
    // ... 60+ countries
  ]
}

Configuration Options

  • enableFileLogging: Enable/disable log file writing
  • enableFileHashing: Write sanitized (hashed) log file for privacy
  • enableExternalLogging: Try SD/USB storage before /tmp
  • enableRoundaboutLatching: Keep roundabout info until exit
  • enableHeartbeat: 2-second BAP resend to override native nav
  • forceImperial: Force miles/mph (overrides DSI/JSON)
  • forceRHD: Force right-hand drive display
  • logFileSize: Log rotation size in MB (1-100)
